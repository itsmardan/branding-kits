 🚀 Using Hugo Templates (Template 1 & 3)

Hugo is a fast and flexible static site generator written in Go.

 📥 Install Hugo

Official install guide: [https://gohugo.io/getting-started/installing/](https://gohugo.io/getting-started/installing/)

 🏃 Run Hugo Site Locally

1. Open the template folder in VS Code.
2. Open a terminal and run:

```bash
hugo serve
```

3. Visit your site at: `http://localhost:1313`

 🔨 Build the Site

```bash
hugo
```

The static site will be generated in the `public/` folder.

---

 🚀 Using Jekyll Template (Template 2)

Jekyll is a static site generator written in Ruby, used for GitHub Pages and many blogs.

 📥 Install Jekyll

Follow the instructions here: [https://jekyllrb.com/docs/installation/](https://jekyllrb.com/docs/installation/)

Make sure Ruby, Bundler, and Jekyll are installed.

 🏃 Run Jekyll Site Locally

```bash
bundle install
bundle exec jekyll serve
```

Visit your site at: `http://localhost:4000`

---

 🌐 Deploying to Netlify

You can host any of the templates for free on Netlify.

 ✅ Steps to Deploy:

1. Push the template folder to a GitHub repository.
2. Go to [https://app.netlify.com/](https://app.netlify.com/)
3. Click "Add new site" → "Import from GitHub"
4. Connect your GitHub account and select the repo.
5. Configure build settings:
   - For Hugo (Template 1 & 3):
     - Build command: `hugo`
     - Publish directory: `public`
   - For Jekyll (Template 2):
     - Build command: `jekyll build`
     - Publish directory: `_site`
6. Click Deploy Site

Your site should be live in seconds!
